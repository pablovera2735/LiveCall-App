LiveCall App



LiveCall App es una plataforma de videollamadas en tiempo real basada en WebRTC, Node.js y Angular. Permite crear y unirse a salas de reunión directamente desde el navegador, con transmisión de audio, video, chat integrado y opciones de control para cada participante. Diseñado como un proyecto educativo y escalable para explorar tecnologías RTC modernas.

✨ Características



&#x20;   Videollamadas en tiempo real mediante WebRTC



&#x20;   Salas de reunión privadas o públicas



&#x20;   Chat integrado en cada sala



&#x20;   Control de audio y video por participante



&#x20;   Compartición de pantalla



&#x20;   Servidor de señalización con WebSockets



&#x20;   Interfaz moderna desarrollada con Angular



&#x20;   Arquitectura escalable para múltiples salas simultáneas



🧱 Tecnologías utilizadas



&#x20;   WebRTC



&#x20;   Node.js



&#x20;   Angular



&#x20;   TypeScript



&#x20;   Docker para despliegue
